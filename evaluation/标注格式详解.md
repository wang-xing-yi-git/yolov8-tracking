# 标注格式详解 & 对比逻辑

## 📋 **总体流程**

```
手动标注 test_annotations.json (地面真实)
       ↓
运行 detect_and_track.py 生成 predictions.json (程序预测)
       ↓
AnomalyEvaluator 逐帧逐对象对比
       ↓
计算混淆矩阵 (TP/FP/FN/TN) 和指标 (准确率/精确度/召回率/F1)
```

### **完整示例**

```json
{

  "frames": [

      "frame_id": 1,

        {
          "track_id": 1,
          "class": "car",

          "reason": "继续位置突跳"
        },
        {
          "track_id": 2,
          "class": "car",
          "is_anomalous": false,
          "reason": "恢复正常速度"
        }
      ]

    {

      "anomalies": [
        {
          "track_id": 1,
          "class": "car",

          "reason": "背景物体，误检"
        },
        {
          "track_id": 3,
          "class": "person",
          "is_anomalous": true,
          "reason": "人物躺倒，宽高比异常"
        }
      ]
    }
  ]
}
```

### **字段说明**

| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| `video` | string | ✓ | 视频文件名 |
| `fps` | number | ✓ | 视频帧率 |
| `frames` | array | ✓ | 所有帧的标注 |
| `frame_id` | number | ✓ | 帧编号 (从1开始) |
| `anomalies` | array | ✓ | 该帧中的所有对象标注 |
| `track_id` | number | ✓ | 追踪ID (从程序输出获取) |
| `class` | string | - | 对象类别 (可选,便于分类评估) |
| `is_anomalous` | boolean | ✓ | 是否异常 (true/false) |
| `reason` | string | - | 异常原因 (可选,便于分析) |

### **标注规则**

#### **车辆 (car/truck/bus/motorbike/bicycle) 异常判断**
```
- is_anomalous = true  ：位置出现突跳 (距离>100px/帧)
- is_anomalous = false ：正常移动,无异常跳跃
```

#### **人物 (person) 异常判断**
```
- is_anomalous = true  ：躺倒/蹲下 (宽高比 > 1.5,即高度>>宽度)
- is_anomalous = false ：正常站立或行走
```

## ⚖️ **3. 对比逻辑 - AnomalyEvaluator.evaluate()**

### **对比流程详解**

```
┌─────────────────────────────────────────────────────────────┐
│ 步骤1: 加载数据                                             │
└─────────────────────────────────────────────────────────────┘

annotations.json (手动标注)
└─> gt_dict = _build_gt_dict()
    {
      frame_1: {track_1: 0, track_2: 1, ...},
      frame_2: {track_1: 1, track_2: 0, ...},
      ...
    }

predictions.json (程序输出)
└─> frames: [{frame_id, predictions: {...}}]

┌─────────────────────────────────────────────────────────────┐
│ 步骤2: 逐帧逐对象对比                                       │
└─────────────────────────────────────────────────────────────┘

for frame_data in predictions['frames']:
    frame_id = frame_data['frame_id']  # 例: 1
    
    for pred_obj in frame_data['predictions']:
        track_id = pred_obj['track_id']  # 例: 2
        pred_anomaly = pred_obj['is_anomalous']  # 例: True
        
        # 从真值字典查询
        true_anomaly = gt_dict[frame_id][track_id]  # 例: True
        
        # 对比并计算混淆矩阵
        if true_anomaly == 1 and pred_anomaly == 1:
            TP += 1  # 真正例 ✓
        elif true_anomaly == 0 and pred_anomaly == 1:
            FP += 1  # 假正例 ✗ (误报)
        elif true_anomaly == 1 and pred_anomaly == 0:
            FN += 1  # 假负例 ✗ (漏检)
        else:  # true_anomaly == 0 and pred_anomaly == 0
            TN += 1  # 真负例 ✓

┌─────────────────────────────────────────────────────────────┐
│ 步骤3: 计算指标                                             │
└─────────────────────────────────────────────────────────────┘

accuracy  = (TP + TN) / (TP + FP + FN + TN)
precision = TP / (TP + FP)
recall    = TP / (TP + FN)
F1_score  = 2 * (precision * recall) / (precision + recall)
```

### **代码对比部分（anomaly_evaluator.py 第44-73行）**

```python
def evaluate(self):
    """执行评估"""
    # 收集所有真值和预测
    true_labels = []
    pred_labels = []
    
    # 转换标注为统一格式
    gt_dict = self._build_gt_dict()  # 🔍 构建真值字典
    
    # 逐帧比较
    for frame_data in self.predictions.get('frames', []):
        frame_id = frame_data['frame_id']
        
        for pred_obj in frame_data.get('predictions', []):
            track_id = pred_obj['track_id']
            pred_anomaly = pred_obj['is_anomalous']  # 🔴 获取程序预测
            
            # 查找对应的真值
            true_anomaly = self._get_ground_truth(gt_dict, frame_id, track_id)  # 🟢 获取人工标注
            
            true_labels.append(true_anomaly)
            pred_labels.append(pred_anomaly)
            
            # 🎯 对比并计算混淆矩阵
            if true_anomaly == 1 and pred_anomaly == 1:
                self.tp += 1
            elif true_anomaly == 0 and pred_anomaly == 1:
                self.fp += 1
            elif true_anomaly == 1 and pred_anomaly == 0:
                self.fn += 1
            else:
                self.tn += 1
    
    # 计算评估指标
    self._calculate_metrics(true_labels, pred_labels)
    
    return self.results
```

## 📊 **4. 实例演示**

### **场景**：一个短视频的标注与对比

**真实发生的帧1-5：**
- 帧1: 车1正常移动 → 标注 `is_anomalous=false`
- 帧2: 车1突跳 → 标注 `is_anomalous=true`
- 帧3: 车1继续跳 → 标注 `is_anomalous=true`
- 帧4: 车1恢复正常 → 标注 `is_anomalous=false`
- 帧5: 行人躺倒 → 标注 `is_anomalous=true`

**程序预测结果：**
- 帧1: 车1预测 `is_anomalous=false` ✓ **TP**
- 帧2: 车1预测 `is_anomalous=true` ✓ **TP**
- 帧3: 车1预测 `is_anomalous=true` ✓ **TP**
- 帧4: 车1预测 `is_anomalous=false` ✓ **TP**
- 帧5: 行人预测 `is_anomalous=false` ✗ **FN** (漏检了躺倒)

### **评估结果**

```
混淆矩阵:
  TP=4, FP=0, FN=1, TN=0

指标:
  准确率 = (4+0)/(4+0+1+0) = 80%
  精确度 = 4/(4+0) = 100%
  召回率 = 4/(4+1) = 80%
  F1分数 = 2*(100%*80%)/(100%+80%) = 88.9%
```
