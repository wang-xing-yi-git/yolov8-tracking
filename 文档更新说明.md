## 🎯 关键知识点总结

### 程序执行的6个主要步骤

```
┌─ 步骤1: 初始化 ─────────────────────────────┐
│ 创建SORT追踪器、生成随机颜色、加载配置      │
└──────────────────────────────────────────────┘
        ↓
┌─ 步骤2: 模型加载 ───────────────────────────┐
│ 检查/下载权重、转移到GPU、初始化数据加载器  │
└──────────────────────────────────────────────┘
        ↓
┌─ 步骤3: 创建推理器 ─────────────────────────┐
│ 实例化DetectionPredictor、启动主循环       │
└──────────────────────────────────────────────┘
        ↓ 循环处理每一帧
┌─ 步骤4: 帧处理（6个子步骤）────────────────┐
│ 4.1 加载帧      4.2 预处理  4.3 推理        │
│ 4.4 后处理      4.5 追踪    4.6 保存        │
└──────────────────────────────────────────────┘
        ↓ 直到视频结束
┌─ 步骤5: 完成处理 ───────────────────────────┐
│ 释放资源、关闭输出视频文件                  │
└──────────────────────────────────────────────┘
```

### SORT追踪算法的4个核心步骤

```
输入: 当前帧检测框(D_t) + 前帧追踪(T_{t-1})
    ↓
[1] 预测阶段 - Kalman滤波器预测现有追踪位置
    ↓
[2] 相似度计算 - 计算检测与预测的IOU矩阵
    ↓
[3] 数据关联 - 匈牙利算法求最优二部匹配
    ↓
[4] 状态更新 - 更新/创建/删除追踪对象
    ↓
输出: 更新后的追踪结果
```

## 🔑 核心代码逻辑速览

### 主程序入口（detect_and_trk.py）
```python
def predict(cfg):
    init_tracker()          # 初始化追踪器
    random_color_list()     # 生成颜色表
    predictor = DetectionPredictor(cfg)
    predictor()             # 启动推理循环
```

### 追踪循环（结构化伪代码）
```python
for frame in video_frames:
    # 4.2 预处理
    tensor = preprocess(frame)
    
    # 4.3 推理
    predictions = model(tensor)
    
    # 4.4 后处理
    detections = postprocess(predictions)
    
    # 4.5 追踪和异常检测
    tracked_objects = tracker.update(detections)
    anomalies = detect_anomalies(tracked_objects)
    
    # SORT算法核心步骤
    predict()           # Kalman预测
    calculate_iou()     # 相似度
    hungarian()         # 匹配关联
    update_tracks()     # 状态更新
    
    # 4.6 保存
    save_frame(output_video)
```

## 💡 重要概念速查

| 概念 | 说明 | 参考位置 |
|------|------|----------|
| **Kalman滤波** | 用恒速模型预测目标位置 | SORT算法部分 |
| **IOU** | 检测框与预测框的重叠比 | 相似度计算 |
| **匈牙利算法** | 求最优二部匹配 | 数据关联 |
| **max_age** | 追踪丢失多少帧后删除 | 配置参数 |
| **min_hits** | 多次检测后才创建追踪 | 配置参数 |
| **NMS** | 非极大值抑制（去重叠） | 后处理步骤 |
